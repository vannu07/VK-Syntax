##  Get Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

**Build:** `npm run build` | **Deploy:** `npm run build && vercel --prod`

##  Structure

```
src/
 pages/          # Home, About, Experience, Projects, Skills, Certifications, Contact
 components/     # Navbar, Footer, AIChatbot, PageBackground
 hooks/          # Custom logic
 data/           # Static content
```
